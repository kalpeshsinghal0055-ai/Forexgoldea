## Deploy on GitHub Pages
1. Upload all files to the repo root (keep the names exactly).
2. Repo → **Settings → Pages** → Source: **Deploy from a branch** → Branch: **main** → **/ (root)** → Save.
3. Your site goes live at `https://<username>.github.io/Forexgoldea/` in ~1 minute.

## Before publishing — replace these
- **Domain:** swap `https://forexgoldea.com/` (and `https://your-domain.com/`) in `index.html` / `blog.html` meta tags for your real URL. On GitHub Pages, use your `github.io` URL or a custom domain.
- **VIP checkout:** the `Get VIP — $99` button links to `#` — point it at your real checkout.
- **Performance numbers:** win rate / profit factor / drawdown are illustrative placeholders — use your own verified figures.
- **Blog posts:** cards are starter templates (`#`) — add real articles.
